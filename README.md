# 旅游信息管理系统 java1066

## 项目概述

旅游信息管理系统是基于 Vue3 和 Node.js 的前后端分离项目。本项目致力于为用户提供便捷的旅游信息管理服务，包含用户管理、景点管理、酒店管理等多个功能模块。

## 技术栈

- 前端：Vue3
- 后端：Node.js、Express框架
- 数据库：Sequelize、MySQL

## 功能模块

- 登录注册
- 用户管理
- 景点管理
- 酒店管理
- 留言管理
- 收藏管理

## 系统角色

- 普通用户：登录、注册、浏览信息、管理个人信息、收藏景点、留言等
- 管理员：用户管理、景点管理、酒店管理、留言管理等

## 运行环境

- 前端：现代浏览器
- 后端：Node.js 环境
- 数据库：MySQL 数据库

## 部署步骤

1. 准备运行环境（Node.js、MySQL）
2. 创建数据库及表结构
3. 部署后端服务
4. 部署前端项目

## 目录结构

```
project
│
├── frontend  # 前端代码
│   ├── public
│   ├── src
│   └── ...
└── backend   # 后端代码
    ├── models
    ├── routes
    ├── controllers
    └── ...
```

## 核心亮点

- 前后端分离架构，便于维护和扩展
- 使用 Vue3，提升前端性能和开发效率
- 采用 Express 框架，简化后端开发流程
- 结合 Sequelize 和 MySQL，实现高效的数据存储和管理
- 多个功能模块，满足不同角色的需求
- 系统稳定，易于部署和运行
